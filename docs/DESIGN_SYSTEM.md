# Design system — Roxo Divine Insight

Identidade **roxo personalizado** alinhada ao web (`#8B5CF6` / `#7C3AED`), fundo quente `#F7F6F2`, highlights lavanda. O FAB central usa gradiente **índigo → violeta** (`#6366F1` → `#8B5CF6`) como na referência.

> Tokens em `lib/core/theme/`, espelhados no app `divine_insight_compass`.

## Paleta (light)

| Token | Hex |
|--------|-----|
| Fundo | `#F7F6F2` |
| Superfície | `#FFFFFF` |
| Superfície suave | `#F5F3FF` |
| Texto primário | `#1B1F23` |
| Texto secundário | `#667085` |
| Acento primário | `#8B5CF6` |
| Acento forte | `#7C3AED` |
| Highlight (containers) | `#EDE9FE` |
| Borda | `#E9E4F0` |
| Sucesso | `#5C8A72` |
| Aviso | `#D97706` |

## Gradientes

- **`brandPrimary`** / **`brandSoft`** — violeta → violeta forte.
- **`fabBible`** — índigo → violeta (botão central Bíblia).
